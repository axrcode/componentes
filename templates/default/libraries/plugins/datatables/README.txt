Esta librería se descargó en el siguiente link(Version para  Bootstrap 3):
https://datatables.net/download/index

Página donde se obtuvo los códigos compatibles: https://www.w3schools.com/cssref/css_entities.asp

Las siguientes caracteres son incompatibles con el ISO-8859-1 del proyecto y se reemplazaron con sus respectivos códigos compatibles:
 - "? BLACK UP-POINTING SMALL TRIANGLE": 	"\25B4"
 - "? BLACK DOWN-POINTING SMALL TRIANGLE":  "\25BE"
 - "+ PLUS SIGN":                           "\002B"
 - "- HYPHEN-MINUS":                        "\002D"
 - "  SPACE":                               "\0020"
 - "? CHECK MARK":                          "\2713"
